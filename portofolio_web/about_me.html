About me

Perkenalkan, nama saya Balqis Hanifa, saya berasal dari Desa Haurgeulis. Alasan saya memilih jurusan Rekayasa Perangkat Lunak (RPL) adalah karena saya tertarik dengan dunia teknologi, khususnya dalam pengembangan software dan aplikasi. Saya ingin belajar lebih banyak tentang pemrograman, desain sistem, dan teknologi komputer agar bisa menjadi seorang programmer atau pengembang aplikasi profesional di masa depan. Saya juga berharap dapat mengembangkan keterampilan saya di sini dan bekerja sama dengan teman-teman untuk mencapai tujuan bersama.

Riwayat sekolah

Saya memulai pendidikan di SDN Haurgeulis di tahun 2014 hingga 2020. Setelah itu, saya melanjutkan ke SMPN 2 Maja dari tahun 2020 hingga 2023. Kini, saya bersekolah di SMKN 1 Maja jurusan Rekayasa Perangkat Lunak (RPL) dengan harapan dapat memperdalam minat saya di bidang teknologi dan pengembangan perangkat lunak.
